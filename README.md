# Trip-Cost-Calculator

This is a  simple Python program to calculate the total cost of a trip, including hotel stay, flight charges, car rental, and additional spending money. 

This project demonstrates modular programming in Python by using separate functions for each cost component. 

## Features - 

Hotel Cost: Calculates cost at $140 per night.

Flight Cost: Returns round-trip flight cost based on destination.

Car Rental Cost: Calculates rental cost at $40/day with discounts: 
- $50 off for rentals of 7+ days

Trip Cost: Combines hotel, flight, car rental, and spending money into a final total.
